**Pattern:** Mirror the toolbar and tag-chip sections from `contacts-table.tsx` but extracted into a standalone component (not embedded inside the table). The filter state is entirely URL-driven — no local state except for the debounced `query` input.

**Search input (300ms debounce):**
```tsx
const [query, setQuery] = React.useState(currentQuery ?? '')
React.useEffect(() => {
  if ((currentQuery ?? '') === query) return
  const timer = setTimeout(() => {
    const params = new URLSearchParams(searchParams.toString())
    if (query) params.set('q', query)
    else params.delete('q')
    params.delete('page')
    router.replace(`${pathname}?${params.toString()}`)
  }, 300)
  return () => clearTimeout(timer)
}, [query])
```
Input: `<Search>` icon prefix, placeholder "Search by name or domain…", `<X>` clear button identical to contacts-table pattern.

**Filter row (flex-row, gap-2):** Five `<Select>` dropdowns side by side, each calling `setParam(key, value)`:

**`setParam` helper** (same as contacts-table):
```typescript
function setParam(key: string, value: string | null) {
  const params = new URLSearchParams(searchParams.toString())
  if (value) params.set(key, value)
  else params.delete(key)
  if (key !== 'page') params.delete('page')
  router.replace(`${pathname}?${params.toString()}`)
}
```
