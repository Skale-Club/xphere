<task type="auto">
  <name>Task 1: AccountCombobox component</name>
  <files>src/components/accounts/account-combobox.tsx</files>
  <action>
Create `src/components/accounts/account-combobox.tsx` as a `'use client'` component.

**Component behavior:**

1. **Input field:** A styled `<Input>` (not a native select) that shows the selected account name or empty. Class: `h-10 text-[13.5px]`.

2. **Dropdown:** A `<div>` positioned absolutely below the input (`absolute z-50 w-full mt-1 rounded-[10px] border border-border bg-bg-secondary shadow-lg max-h-[200px] overflow-y-auto`). Visible when `inputValue.length > 0 || isOpen`.

3. **Debounced search (300ms):** When `inputValue` changes, wait 300ms then call `getAccounts({ q: inputValue, pageSize: 20 })`. Show a loading indicator during fetch (a simple `<Loader2 className="h-3.5 w-3.5 animate-spin">` inside the dropdown).

4. **Dropdown items:** For each account in results:
```tsx
<button
  key={account.id}
  type="button"
  onClick={() => handleSelect(account.id, account.name)}
  className="w-full text-left px-3 py-2 text-[13px] hover:bg-bg-tertiary flex items-center gap-2"
>
  <Building2 className="h-3.5 w-3.5 text-text-tertiary shrink-0" />
  <span className="font-medium text-text-primary truncate">{account.name}</span>
  {account.domain && (
    <span className="ml-auto text-[11.5px] text-text-tertiary shrink-0">{account.domain}</span>
  )}
</button>
```

5. **"Create new company" option** — always show at the bottom of the dropdown (even when results are empty):
```tsx
<button
  type="button"
  onClick={() => setShowCreate(true)}
  className="w-full text-left px-3 py-2 text-[12.5px] text-accent hover:bg-accent-muted/30 flex items-center gap-2 border-t border-border-subtle"
>
  <Plus className="h-3.5 w-3.5" />
  Create new company "{inputValue || '…'}"
</button>
```

6. **Inline quick-create flow:** When `showCreate === true`, replace the dropdown with a tiny inline form:
```tsx
<div className="absolute z-50 w-full mt-1 rounded-[10px] border border-border bg-bg-secondary shadow-lg p-3 space-y-2">
  <p className="text-[12px] font-medium text-text-primary">New company</p>
  <Input
    value={createName}
    onChange={(e) => setCreateName(e.target.value)}
    placeholder="Company name"
    autoFocus
    className="h-9 text-[13px]"
  />
  <div className="flex items-center gap-2 justify-end">
    <Button type="button" size="sm" variant="ghost" onClick={() => setShowCreate(false)}>Cancel</Button>
    <Button
      type="button"
      size="sm"
      disabled={!createName.trim() || isCreating}
      onClick={handleCreate}
    >
      {isCreating ? 'Creating…' : 'Create'}
    </Button>
  </div>
</div>
```
`handleCreate` calls `createAccount({ name: createName.trim() })`, on success: calls `onChange(result.data.id, result.data.name)`, sets `inputValue` to `result.data.name`, closes dropdown and quick-create panel, shows `toast.success('Company created')`.

7. **Clearing:** When `inputValue` is cleared, call `onChange(null, null)`.

8. **Click-outside to close:** Use a `React.useRef` on a wrapper div + `useEffect` with `document.addEventListener('mousedown', handler)`.

9. **State variables:** `inputValue: string`, `results: AccountWithCounts[]`, `isLoading: boolean`, `isOpen: boolean`, `showCreate: boolean`, `createName: string`, `isCreating: boolean`.

**Exports:** `export function AccountCombobox(...)` — named export.

**Imports:** `React`, `getAccounts`, `createAccount` from `@/app/(dashboard)/accounts/actions`, `AccountWithCounts` from `@/lib/accounts`, `Input`, `Button` from shadcn, `Building2`, `Plus`, `Loader2` from lucide-react, `toast` from sonner, `cn`.

**Step 2 — Update ContactForm component:**
1. Import `AccountCombobox` from `@/components/accounts/account-combobox`

3. Replace the Company `<Field>` block:

```tsx
<Field label="Company" htmlFor="contact-company">
  <AccountCombobox
    value={accountId}
    onChange={(id, name) => {
      setValue('account_id', id, { shouldDirty: true })
      // Keep legacy company text field in sync (hidden but still part of submit)
      if (name !== null) setValue('company', name, { shouldDirty: true })
    }}
    defaultAccountName={defaultValues?.company ?? undefined}
  />
</Field>
```

**Step 3 — Update useForm defaultValues:**
Add `account_id: defaultValues?.account_id ?? null` to the `useForm` defaultValues object.

**Important:** Do NOT break any other existing part of the form. The `source`, `tags`, `notes`, `phone`, `email`, `name` fields remain unchanged.
