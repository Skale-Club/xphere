**Query 1 — refresh:**
```sql
SELECT public.refresh_contact_duplicate_audit();
```
Expected: returns void (single row, NULL).

**Query 2 — high-level distribution:**
```sql
SELECT
  match_type,
  count(*) AS clusters,
  sum(cluster_size)::int AS total_dupe_rows,
  max(cluster_size)::int AS largest_cluster,
  min(cluster_size)::int AS smallest_cluster,
  round(avg(cluster_size)::numeric, 2) AS avg_cluster_size
FROM public.contact_duplicate_audit
GROUP BY match_type
ORDER BY match_type;
```

**Query 3 — cluster size histogram:**
```sql
SELECT
  match_type,
  CASE
    WHEN cluster_size = 2 THEN '2'
    WHEN cluster_size BETWEEN 3 AND 5 THEN '3-5'
    WHEN cluster_size BETWEEN 6 AND 10 THEN '6-10'
    WHEN cluster_size BETWEEN 11 AND 25 THEN '11-25'
    ELSE '26+'
  END AS bucket,
  count(*) AS clusters
FROM public.contact_duplicate_audit
GROUP BY match_type, bucket
ORDER BY match_type, bucket;
```

**Query 4 — per-org distribution (top 20 orgs by duplicate count):**
```sql
SELECT
  org_id,
  count(*) FILTER (WHERE match_type='phone') AS phone_clusters,
  count(*) FILTER (WHERE match_type='email') AS email_clusters,
  sum(cluster_size) FILTER (WHERE match_type='phone') AS phone_dupe_rows,
  sum(cluster_size) FILTER (WHERE match_type='email') AS email_dupe_rows
FROM public.contact_duplicate_audit
GROUP BY org_id
ORDER BY (count(*)) DESC
LIMIT 20;
```

**Query 5 — top 10 largest phone clusters (sample for Phase 106 UI sizing):**
```sql
SELECT cluster_id, org_id, normalized_value, cluster_size, array_length(contact_ids, 1) AS arr_len
FROM public.contact_duplicate_audit
WHERE match_type = 'phone'
ORDER BY cluster_size DESC
LIMIT 10;
```
Assert: `cluster_size = arr_len` for every row (consistency check of refresh function).

**Query 6 — top 10 largest email clusters:**
```sql
SELECT cluster_id, org_id, normalized_value, cluster_size, array_length(contact_ids, 1) AS arr_len
FROM public.contact_duplicate_audit
WHERE match_type = 'email'
ORDER BY cluster_size DESC
LIMIT 10;
```

**Query 7 — Pitfall 7 oversized normalized values:**
```sql
SELECT match_type, length(normalized_value) AS len, normalized_value, cluster_size
FROM public.contact_duplicate_audit
WHERE length(normalized_value) > 20
ORDER BY len DESC
LIMIT 20;
```
