```typescript
export async function updateGoogleContact(
  params: Record<string, unknown>,
  ctx: ActionContext
): Promise<string>
```

**Params interface (internal):**
```typescript
interface UpdateContactParams {
  email: string   // required — used to find the contact
  name?: string
  phone?: string
  company?: string
  notes?: string
  [key: string]: unknown
}
```

**Implementation:**

1. `const tokens = await resolveGoogleCredentials(ctx)`
2. `const { email, name, phone, company, notes } = params as UpdateContactParams`
3. If `!email` → throw `new Error('update_contact requires email param to locate the contact')`

**Step A — Search for the contact:**
4. Build search URL: `new URL('https://people.googleapis.com/v1/people:searchContacts')`, set params: `query=email`, `readMask=names,emailAddresses,phoneNumbers,organizations,biographies`, `pageSize=1`
5. Call `callWithRefresh(tokens.access_token, tokens.refresh_token, tokens.integrationId, ctx, (token) => fetch(searchUrl.toString(), { headers: { Authorization: \`Bearer \${token}\` }, cache: 'no-store' }))`
6. If not ok → throw `new Error(\`People API search error \${searchRes.status}\`)`
7. `const searchData = await searchRes.json() as { results?: Array<{ person: GooglePerson }> }`
8. If `!searchData.results?.length` → throw `new Error(\`Contact not found for update: \${email}\`)`
9. `const person = searchData.results[0].person`
10. `const { resourceName, etag } = person`

**Step B — Build update body:**
11. Build the `updatePersonFields` mask dynamically:
    ```typescript
    const fieldMap: Record<string, string> = {
      name: 'names', email: 'emailAddresses', phone: 'phoneNumbers',
      company: 'organizations', notes: 'biographies',
    }
    // email is used for lookup — only include in mask if also updating email
    const mask = Object.keys(params)
      .filter(k => fieldMap[k] && params[k])
      .map(k => fieldMap[k])
      .join(',')
    ```
12. Build the update body object with top-level `etag`:
    ```typescript
    const updateBody: Record<string, unknown> = { etag }
    if (name) {
      const parts = name.trim().split(/\s+/)
      updateBody.names = [{ givenName: parts[0] ?? '', familyName: parts.slice(1).join(' ') }]
    }
    if (email) updateBody.emailAddresses = [{ value: email }]
    if (phone) updateBody.phoneNumbers = [{ value: phone }]
    if (company) updateBody.organizations = [{ name: company }]
    if (notes) updateBody.biographies = [{ value: notes }]
    ```

**Step C — PATCH the contact:**
13. Build PATCH URL: `\`https://people.googleapis.com/v1/\${resourceName}:updateContact?updatePersonFields=\${mask}\``
14. Call `callWithRefresh(tokens.access_token, tokens.refresh_token, tokens.integrationId, ctx, (token) => fetch(patchUrl, { method: 'PATCH', headers: { Authorization: \`Bearer \${token}\`, 'Content-Type': 'application/json' }, body: JSON.stringify(updateBody), cache: 'no-store' }))`
15. If not ok → `const text = await patchRes.text().catch(() => \`status \${patchRes.status}\`); throw new Error(\`People API error \${patchRes.status}: \${text}\`)`
16. Return `\`Google contact updated. Resource: \${resourceName}\``

**Imports:**
- `import type { ActionContext } from '@/lib/action-engine/execute-action'`
- `import { resolveGoogleCredentials, callWithRefresh } from './credentials'`

CRITICAL: The etag MUST be the top-level field in the PATCH body — omitting it causes 400 failedPrecondition. Use the etag from the searchContacts result (no separate GET needed). No newlines in return string. TypeScript strict: no `any`.
