Create `src/lib/google-contacts/credentials.ts` with the following implementation:

**Exports:**
- `interface GoogleOAuthTokens` — `{ access_token: string; refresh_token: string; google_email?: string; integrationId: string }`
- `async function resolveGoogleCredentials(ctx: ActionContext): Promise<GoogleOAuthTokens>`
- `async function callWithRefresh<T>(access_token: string, refresh_token: string, integrationId: string, ctx: ActionContext, apiFn: (token: string) => Promise<Response>): Promise<Response>`

**resolveGoogleCredentials implementation:**
1. Query `ctx.supabase.from('integrations').select('id, encrypted_api_key').eq('organization_id', ctx.organizationId).eq('provider', 'google_contacts').single()`
2. If `error || !row` → throw `new Error('Google Contacts not connected for this org. Connect via /integrations.')`
3. `const blob = JSON.parse(await decrypt(row.encrypted_api_key))` — typed as `{ access_token: string; refresh_token: string; google_email?: string }`
4. Return `{ access_token: blob.access_token, refresh_token: blob.refresh_token, google_email: blob.google_email, integrationId: row.id }`

**callWithRefresh implementation:**
1. `let res = await apiFn(access_token)` — call the API with the current token
2. `if (res.status !== 401) return res` — if not 401, return immediately (success or other error)
3. On 401: build refresh POST body with `URLSearchParams({ client_id: process.env.GOOGLE_CLIENT_ID!, client_secret: process.env.GOOGLE_CLIENT_SECRET!, refresh_token, grant_type: 'refresh_token' })`
4. POST to `https://oauth2.googleapis.com/token` with `Content-Type: application/x-www-form-urlencoded` and `cache: 'no-store'`
5. If refresh response not ok → throw `new Error('Google token refresh failed — reconnect via /integrations.')`
6. Parse refresh response as `{ access_token: string; expires_in: number }`
7. Build new blob: `{ access_token: refreshData.access_token, refresh_token, google_email: undefined }` — DO NOT use the refresh response's refresh_token (Google does NOT return one; reuse the original)
8. `newExpiry = Date.now() + refreshData.expires_in * 1000`
9. Persist: `await ctx.supabase.from('integrations').update({ encrypted_api_key: await encrypt(JSON.stringify(newBlob)), config: { token_expiry: newExpiry } }).eq('id', integrationId)`
10. Retry: `return apiFn(refreshData.access_token)`

**Imports needed:**
- `import type { ActionContext } from '@/lib/action-engine/execute-action'`
- `import { encrypt, decrypt } from '@/lib/crypto'`
- `import type { Json } from '@/types/database'` (for the config update cast)

TypeScript strict mode: all types must be explicit. No `any`.

**Signature:**
```typescript
export async function findGoogleContact(
  params: Record<string, unknown>,
  ctx: ActionContext
): Promise<string>
```

**Internal types:**
```typescript
interface GooglePerson {
  resourceName: string
  etag: string
  names?: Array<{ displayName: string; givenName: string; familyName: string }>
  emailAddresses?: Array<{ value: string }>
  phoneNumbers?: Array<{ value: string }>
}
```

**Implementation steps:**
1. `const tokens = await resolveGoogleCredentials(ctx)`
2. `const query = String(params.email ?? params.phone ?? '')`
3. If `!query` → throw `new Error('find_contact requires email or phone param')`
4. Build URL: `new URL('https://people.googleapis.com/v1/people:searchContacts')`, set params: `query={query}`, `readMask=names,emailAddresses,phoneNumbers`, `pageSize=10`
5. Call `callWithRefresh(tokens.access_token, tokens.refresh_token, tokens.integrationId, ctx, (token) => fetch(url.toString(), { headers: { Authorization: \`Bearer \${token}\` }, cache: 'no-store' }))`
6. If `!res.ok` → throw `new Error(\`People API error \${res.status}\`)`
7. `const data = await res.json() as { results?: Array<{ person: GooglePerson }> }`
8. `const results = data.results ?? []`
9. If `results.length === 0` → return `'No contact found matching that query.'`
10. Build summary from `results[0].person`:
    - `displayName = person.names?.[0]?.displayName ?? ''`
    - `email = person.emailAddresses?.[0]?.value ?? ''`
    - `phone = person.phoneNumbers?.[0]?.value ?? ''`
    - `const parts = [displayName, email, phone].filter(Boolean)`
    - `const summary = parts.join(' | ')`
    - `const suffix = results.length > 1 ? \` (1 of \${results.length} matches)\` : ''`
11. Return `\`Found: \${summary}\${suffix}\``

**Imports:**
