# Requirements: v2.0 Multi-Bot Platform — Observability

**Workstream:** phase-40-observability
**Created:** 2026-05-17

---

## OBS — Agent Observability

- [ ] **OBS-04:** Per-agent metrics widget on `/dashboard/agents/[id]` showing invocation count, p50/p95 latency, total cost, and tool-call success rate over 24h / 7d / 30d windows
- [ ] **OBS-05:** Per-org cost ticker on `/dashboard` with 1h / 24h / 7d totals and `% of daily cap consumed`; alert badge when consumption ≥80% of cap
- [ ] **OBS-06:** `/dashboard/conversations/[id]` renders a delegation tree (collapsible nested invocations) with cost + latency annotated per node
- [ ] **OBS-07:** `/dashboard/agents/[id]/invocations` lists recent invocations filterable by status, cost, and error; clicking an invocation opens its delegation tree view
- [ ] **OBS-08:** Chat-area assistant messages show an agent badge identifying which agent produced each message
